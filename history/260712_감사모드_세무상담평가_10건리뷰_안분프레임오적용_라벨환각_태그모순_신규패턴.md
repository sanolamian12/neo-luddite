## 한 줄 요약
선행 문서(260711)의 반환 포맷·반복 오류 패턴 사전을 그대로 이어받아, 이번 채팅에서 **병의원 세무 상담 AI의 Q&A 10건**을 문장별로 리뷰했다. 각 건 WebSearch로 실제 조문·최신 개정 교차확인 후 **① 문장별 코멘트 ② 세션 평가(문장력·법률정확성 각 5점) ③ 세션 전체 평가 의견** 3종을 반환. 이번 세션의 핵심 발견은 **선행 P1~P9 패턴이 그대로 재현**되면서 **세 가지 신규 패턴이 뚜렷해진 것**: ⑩ "업무/사적 사용 안분" 프레임을 안분 대상이 아닌 사안에 오적용, ⑪ 법해석론 라벨("문언해석·목적론해석·엄격해석") 남발, ⑫ "전부 인정 ↔ 인정 0/0원" 자기모순 태그. 아래 §2가 이 문서의 핵심 자산(선행 사전의 증보판).

## 1. 반환 포맷 (고정) — 선행 문서·메모리 [[feedback_auditor_review_format]] 그대로

사용자가 "질문 + AI 답변" 세트를 올리면 3종 반환. ① 문장별 코멘트(🔴법적해석오류 / 🔴문법오류 / 🟡제안사항, 문장당 1~2줄 태그+근거) ② 세션 평가(문장력·법률정확성 각 ★5점) ③ 세션 전체 평가 의견(결론 요지 + 올바른 답변 방향 재구성 + 되물을 정보 + 인정금액·리스크 태그 정정).

**규칙:** 각 건 독립시행(타 세션 비교 금지, 단 같은 세션 내 턴 간 일관성 지적은 허용) · 법령 지적은 반드시 WebSearch로 실제 조문·최신 개정 교차확인 후 출처 링크 첨부.

## 2. 신규 오류 패턴(P10~P12) + 선행 패턴 재확인 ★ (재사용 핵심)

선행 사전 P1~P9(지출유형 라우팅 실패·템플릿 환각·접대비/적격증빙 오적용·가짜조문·근로소득 누락·한도 방향오류·성급단정·태그 부적합)에 더해, 이번 세션에서 **아래 3종이 반복 패턴으로 확정**됨. 새 리뷰 시 P1~P12를 함께 스캔.

| # | 패턴 | 증상 | 올바른 프레임 |
|---|---|---|---|
| **P10** | **"업무/사적 사용 안분" 프레임 오적용** | 차량·겸용부동산의 안분 로직("업무사용 비율만큼 인정, 개인적 사용분 부인, 100% 업무사용 입증 필요")을 **안분 대상이 아닌 사안**에 반복 적용. 이번 세션 4건(임대료·개원컨설팅·개인명의 장비·AI구독료)에서 출현 | 사안의 진짜 축(시가·부당행위계산 / 비용·자산 구분 / 명의 불일치 / 필요경비 불산입)으로 재정의. "100% 업무입증" 요구는 대개 논점 회피의 신호 |
| **P11** | **법해석론 라벨 남발** | 상담 문장마다 "문언해석·목적론해석·체계적해석·실질과세원칙·입증책임·신의성실원칙·엄격해석" 같은 **법학방법론 태그를 기계적으로 부착**(실질 내용 없이 형식만). P2 라벨 환각의 변형 | 라벨 전부 삭제. 사안별 조문·결론만 간결히 |
| **P12** | **"전부 인정 ↔ 인정 0/0원" 태그 자기모순** | 결과 태그가 "**전부 인정** · **인정 0/0원**"으로 동시 표기되거나, 질문 전제와 **정반대 방향**(개인적 사용인데 "전부 인정") | 판정이 성립하지 않는 질문(방법 문의 등)엔 "정보 필요"로 보류. 인정금액 표기와 판정 라벨 정합 |

**이번 세션에서 다시 확인된 선행 패턴:**
- **P1(지출유형 라우팅 회귀)** — "어떤 종류의 지출인지(차량·접대·통신·복리후생 등) 알려달라"로 **판단을 통째로 회피**. 순수 표본 3건: **로고 제작비 / 의료장비 업그레이드 / 가족 직원 과다급여**. (질문이 명확한 회계·세무 판단인데 지출엔진으로 회귀)
- **P5(가짜/오인용 조문·판례)** — **소득세법 §35**(실제로는 접대비=기업업무추진비 조항)를 필요경비 일반근거로 반복 오인용: **임대료·개인명의 장비·AI구독료** 3건. 광고경품 건은 **소득세법 시행령 §35**를 광고선전비 근거로 오인용. **가짜 판례 "조심2025구1960"** 재출현(가족 명의 계좌 건).
- **P4(적격증빙 부인 오적용)** — **인건비(급여)에 적격증빙·증빙불비가산세(2%) 논리**를 갖다 붙임(가족 명의 계좌 건). 급여는 적격증빙 대상이 아님.
- **P2(템플릿 환각)** — "원장 개인 운동·회원권 명의"가 **직원 피부시술** 답변에 오염 출현.
- **P8(성급 단정·되묻기 부재)** — 거의 전 세션에서 **개인사업자/의료법인 분기**를 안 묻고 단정.

## 3. 이번 세션 검증 법령 미니 KB (WebSearch 확인, 재리서치 절약용)

선행 문서 §3의 증보. 새 세션에서도 재확인 권장하나 출발점으로 사용 가능.

- **특수관계인 부동산 임대 = 적정 임대료는 "시가"**(소득세법 §41 부당행위계산·시행령 §98 / 법인이면 법인세법 §52·시행령 §89). 시가 산정 순서 ① 유사 임대사례가액(주변 시세) → ② 감정평가액 → ③ 법정산식 **(부동산 시가 × 50% − 임대보증금) × 정기예금이자율**. 예: 시가 100억·보증금 3억 → (50억−3억)×2.9% ≈ 1억 3,630만원/년. **발동요건: 시가와 거래가액 차액이 3억원 이상 또는 시가의 5% 이상.** 저가임대→임대인 시가과세, 고가임대→임차인 초과분 부인.
- **개인사업자 본인 건물 자가사용** = 자기에게 지급하는 임차료는 필요경비 불산입, **감가상각비(건물 기준내용연수 40년)로만** 비용화. → "적정 임대료" 문제 자체가 성립 안 함(임대인=임차인).
- **직원에게 무상 제공한 현물(피부시술 등)** = 근로 대가로 제공되는 모든 금품·물품은 **근로소득**(소득세법 §12 비과세목록에 없음) → **원천징수**. 단 전 직원 균등·사회통념상 복리후생 범위면 급여 제외 여지.
- **용역의 무상공급 = 부가세 과세대상 아님**(부가세법 §12②). "사업장 내 사용인에게 무상 제공하는 용역"은 명시적으로 비과세. **유일 예외 = 특수관계인 간 사업용 부동산 임대용역**. → 미용시술 무상제공에 자가공급/사업상 증여로 부가세 물리는 접근은 오류.
- **내부적으로 창출한 브랜드·상표·로고 = 무형자산 인식 금지**(K-IFRS 제1038호 문단 63, 일반기업회계기준 제11장) → 제작비는 **당기 비용(광고선전비/지급수수료)**. **상표권으로 출원·등록해 법적 권리를 취득하거나 외부 매입**하면 무형자산(산업재산권)으로 계상·상각.
- **개업(개원) 전 지출** = 사업관련성이 명확하면 필요경비 인정. 컨설팅이 **인테리어 설계 등 특정 자산 취득에 부수**되면 그 자산 취득원가에 산입(자본화·감가상각). **사업자등록 전 매입세액**은 공급시기가 속한 **과세기간 종료 후 20일 이내에 등록 신청** 시 공제(그 전엔 대표자 주민등록번호로 세금계산서 수취).
- **광고선전비 vs 기업업무추진비(접대비)** = 상대방이 **불특정 다수 → 광고선전비(금액 무관 전액 손금)**, **특정인 → 접대비**. "개당 3만원 이하 제외, 특정인 연 5만원 한도"(법인세법 시행령 §19)는 **특정인 기증 물품**에만 적용. **경품 당첨금 = 수령자의 기타소득**(소득세법 §21), 지급자 원천징수 의무, 건별 기타소득 5만원 이하는 과세최저한.
- **자본적 지출 vs 수익적 지출(수선비)** = 내용연수 연장·가치 증가(개조·개량·확장·증설, 용도변경) → **자본적(자산·감가상각)**. 원상회복·능률유지 → **수익적(수선비·당기비용)**. **소액수선비 특례**(법인세법 시행령 §31③): 개별 자산별 수선비 **600만원 미만** 또는 직전 장부가액의 **5% 미만** 또는 3년 미만 주기적 수선이면 자본적 지출이라도 **수선비로 비용처리 허용**.
- **개인 명의 계약 자산을 사업 사용** = **실질과세(국세기본법 §14)** — 실질이 사업 사용이면 비용 인정 여지. 단 **세금계산서 명의가 사업자와 다르면 매입세액공제 불가**. **결정적 분기: 개인사업자(대표=사업자 본인, 인정 여지 큼) vs 의료법인(별개 인격 → 매입세액공제 불가·대표자 가지급금·부당행위계산부인, 명의 이전 필요).**
- **가족 직원 급여** = 실제 근로 제공 시 인건비 인정(소득세법 시행규칙: 사업에 직접 종사하는 배우자·부양가족 포함, 가족이라는 이유만으로 가공경비 아님). **동일 직위 제3자보다 과다 지급분은 부당행위계산부인(소득세법 §41)으로 손금 부인**, 판단 기준은 동일 직급 다른 직원 급여. **미근무 가족 등록 = 가공인건비 → 세무조사 수년치 추징.**
- **인건비(급여)는 적격증빙(세금계산서·계산서·신용카드·현금영수증) 수취 대상이 아님.** **원천징수영수증·급여대장·지급명세서 제출로 증빙 갈음.** 증빙불비가산세(2%)도 인건비엔 무관. → 가족 명의 계좌 급여의 진짜 리스크는 적격증빙이 아니라 **차명계좌·실질귀속·소득분산/증여·가공인건비 소명**.
- **개인적 사용 지출 = 필요경비 불산입**(소득세법 §33: 가사관련경비 §33①5호, 업무무관 지출 §33①13호). 필요경비 일반규정은 §27(총수입금액에 대응하는 통상적 비용). **소득세법 §35는 접대비 조항이므로 필요경비 일반근거로 인용하면 오류(P5).** 사업용 카드 결제해도 실질이 개인사용이면 부인.

## 4. 이번 세션에서 리뷰한 10건 (순서대로)

각 건 대부분 **법률정확성 1~2/5, 문장력 2~3/5**. 지배적 오류 태그와 점수만 요약.

1. **원장 개인건물 → 병원 임대, 적정 임대료 판단** — 정답축(시가·부당행위계산부인) 전면 누락, §35 오인용(P5), 업무/사적 안분 오프레임(P10), 개인/법인 분기 무시 / 문장력 2·법률 1
2. **직원 무료 피부시술 세무처리** — 근로소득(P6)·부가세(용역 무상공급) 논점 전멸, P2 환각("원장 개인 운동·회원권"), **질문의 "직원들에게"를 "원장 단독 수혜"로 사실 뒤집기** / 문장력 2·법률 1
3. **로고 제작비 비용/자산** — **P1 순수 표본**(판단 전무, 차량·접대·통신으로 회귀). 정답=내부창출 브랜드 무형자산 인식 금지 / 문장력 2·법률 2
4. **개원 컨설팅 비용 전액 비용처리?** — 안분 오프레임(P10), 자기모순(조건부↔확정), **태그 모순 "전부 인정·인정 0원"(P12)**, 개업전 지출·매입세액 누락 / 문장력 2·법률 2
5. **광고 목적 경품 지급** — 방향(불특정다수→광고선전비)·의료법 환자유인 경고는 ✅. 다만 **시행령 §35 오인용(P5)**, "1인당 3만원 이하" 요건 적용대상 혼동, 수령자 기타소득·부가세 누락 / 문장력 3·법률 2
6. **의료장비 업그레이드 수선비/자산** — **P1 순수 표본**(판단 전무). 정답=자본적/수익적 지출 구분 + 소액수선비 특례(600만원) / 문장력 2·법률 2
7. **대표원장 개인명의 계약 장비를 병원 사용** — §35 오인용(P5), 안분 오프레임(P10), **명의 불일치·개인/법인·매입세액공제 논점 전멸**, 태그 모순(P12) / 문장력 2·법률 1
8. **가족 직원 과다 급여의 위험 요소** — **P1 순수 표본**(위험을 하나도 제시 못함). 정답=부당행위계산부인+가공인건비 리스크 / 문장력 2·법률 2
9. **가족 명의 계좌로 급여 지급** — **P4 적격증빙 오적용**(인건비에 세금계산서·증빙불비가산세 2%), **P5 가짜 판례 "조심2025구1960"**, 진짜 논점(차명·실질귀속) 회피 / 문장력 2·법률 1

## 5. 새 세션에서 이어가는 법

1. 이 문서 + 선행 [260711 문서](260711_감사모드_세무상담평가_문장별코멘트_세션평가_반복오류패턴.md) + 메모리 [[feedback_auditor_review_format]] 를 먼저 읽는다.
2. "질문 + AI 답변" 세트가 오면 → §1 포맷(3종)으로 반환.
3. 리뷰 시 **P1~P12**를 함께 스캔. 특히 이번 세션에서 굳어진 신호:
   - "어떤 종류의 지출인지 알려달라"(차량·접대·통신·복리후생) → **P1 회귀, 판단 회피**.
   - "소득세법 §35" 인용 → **P5(§35=접대비 조항, 필요경비 일반근거 아님)**. "조심2025구1960" → **P5 가짜 판례**.
   - "업무사용 비율만큼 인정 / 100% 업무사용 입증" → **P10, 대개 논점 회피**. 사안이 정말 안분 대상인지부터 판별.
   - "문언해석·목적론해석·엄격해석" 라벨 → **P11 삭제 대상**.
   - "전부 인정 · 인정 0/0원" → **P12 모순 태그**.
4. **개인사업자 vs 의료법인 분기**를 거의 모든 사안에서 먼저 되물어야(P8). 답이 통째로 갈리는 1순위.
5. §3 미니 KB를 출발점으로 삼되 **WebSearch로 조문·최신 개정 재확인** 후 출처 링크 첨부.
6. 세션 전체 평가 의견은 **독립시행**(타 세션 비교 금지), "올바른 답변 방향" + 되물을 정보 + 태그 정정 포함.
7. 점수: 틀린 법 진술 없이 회피만 했으면 법률정확성 2점대, 조문 오인용·환각·정반대 오류면 1점.
