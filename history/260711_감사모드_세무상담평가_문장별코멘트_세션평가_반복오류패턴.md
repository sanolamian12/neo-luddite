# 감사 모드(auditor) 세무 상담 평가 — 리뷰 워크플로우 + 반복 오류 패턴 사전

## 한 줄 요약
사용자(auditor 역할, 계정 `평가자3`)가 "상담 평가 > 진행중" 화면에서 **병의원 세무 상담 AI**의 Q&A를 문장별로 검토. 사람이 "질문 + AI 답변" 세트를 붙이면, 나는 도메인 리서치(WebSearch로 실제 조문·최신 개정 교차확인) 후 **① 문장별 코멘트 ② 세션 평가(2점) ③ 세션 전체 평가 의견** 3종을 반환한다. 이 채팅에서 14개 세션을 리뷰했고, 감사 대상 AI에서 **동일한 오류 패턴이 반복**됨을 확인했다(아래 "반복 오류 패턴 사전"이 이 문서의 핵심 자산). 별도로 진행중 목록 UI(진행도 2컬럼 분리)를 수정·배포함.

## 1. 반환 포맷 (고정) — 메모리 [[feedback_auditor_review_format]]

사용자가 "질문 + AI 답변" 세트를 올리면 아래 3종을 반환한다.

**① 문장별 코멘트** — 문제 문장을 anchor로 인용 + 유형 태그 + 근거. 유형 3종:
- 🔴 **법적 해석 오류** — 참고 법령/법 해석 자체가 틀림.
- 🔴 **문법적 오류** — 비문·앞뒤 논리 모순·자기모순.
- 🟡 **제안사항** — 세부정보를 되물었어야 하는 등 상담 행동 미흡.

**② 세션 평가** — 두 팩터를 각 5점 만점(★) 레이팅:
- **문장력** — 비문·환각·자기모순 여부, 상담 톤.
- **법률적 정확성** — 조문·해석·핵심 논점 정확성.

**③ 세션 전체 평가 의견** — 결론 요지 + "올바른 답변 방향"(정답 재구성) + 되물었어야 할 핵심 정보 + 인정금액·리스크 태그 정정 방향.

**중요 규칙 (사용자 확정):**
- 각 건은 **독립시행**으로 평가한다. 세션 전체 평가 의견에 **다른 세션/과거 답변과 비교하는 표현을 넣지 말 것**("앞 세션보다", "동일 템플릿" 등 금지). ※단, 같은 세션 내 여러 턴 간 일관성 지적은 허용.
- 법령 지적은 반드시 WebSearch로 **실제 조문·최신 개정**까지 교차확인 후 근거·출처(markdown 링크)로 제시.
- 문장별 코멘트 분량은 "지금 정도"(문장당 1~2줄 태그+근거)가 적당.

## 2. 감사 대상 AI의 반복 오류 패턴 사전 ★ (재사용 핵심)

감사 대상 상담 AI는 "지출유형 판정 엔진" 위에 얹혀 있어, 그 프레임을 벗어나는 질문에서 아래 패턴이 **반복** 출현한다. 새 세션에서 리뷰할 때 이 패턴을 먼저 의심하면 빠르다.

| # | 패턴 | 증상 | 올바른 프레임 |
|---|---|---|---|
| P1 | **지출유형 라우팅 실패/거부** | "'○○' 항목은 아직 병의원 비용 판정 규칙에 포함되지 않았습니다. 현재 지원하는 지출 유형은 업무용승용차·임차료·접대성지출·광고선전비·통신비·복리후생비·출장비·소프트웨어구독·가사관련비…" 하고 **질문을 거부**하거나, 답변 중 "어떤 종류의 지출인지(차량·접대·통신·복리후생 등) 알려달라"고 **되물으며 회귀** | 자산매각·재고·부가세·기부금·세액공제·노무 등은 애초에 "지출 판정" 대상이 아님. 사안 성격부터 재정의 |
| P2 | **템플릿 환각(오염)** | 질문과 무관한 다른 사안 답변이 통째로 붙음. 특히 **"원장 개인 운동"·"회원권 명의"·"업무 전용 회선 2대 중 1대"·"업무일/관광일 안분(학회 출장)"** 문구가 엉뚱한 곳에 출현 | 해당 문장 전부 무효. 질문 사안으로 재작성 |
| P3 | **접대비 부인 오적용** | "구매/기부 상대방이 거래처가 아니어서 개인적 지출로 부인", 접대 상대방·목적·연 한도 증빙 요구 | 재고 폐기손실·기부금·매출할인 등에 접대비 논리는 무관 |
| P4 | **적격증빙 부인 오적용** | "적격증빙(세금계산서·계산서·신용카드·현금영수증) 미보유 → 손금부인 + 증빙불비가산세(2%)" | 재고 폐기·차량 겸용 안분 등은 적격증빙이 아니라 폐기입증/운행기록부 문제 |
| P5 | **가짜/오인용 조문·판례** | 거의 모든 답변이 **"소득세법 §35"**를 만능 근거로 인용(실제 §35는 무관). 판례 **"조심2025구1960"** 반복 인용 — 조세심판원 사건번호 체계("조심YYYY-지역-####")와 형식 불일치(‘구’는 행정소송 코드)라 **실재 의심** | 사안별 정확 조문으로 교체 |
| P6 | **근로소득 과세 누락** | 직원 명절선물·떡값·자녀학자금을 "복리후생비 인정"만 하고, **수령 직원의 근로소득 과세·원천징수**를 빠뜨림 | 직원에게 준 현금·현물·학자금은 원칙적으로 근로소득(소득세법 §12 비과세목록에 없음) |
| P7 | **한도/특례 방향 오류·누락** | 접대비/업무용승용차 등의 **한도·특례를 정반대로 판단**하거나 누락. "전액 인정" 남발 | 접대비 한도(중소 기본 3,600만+수입금액별 0.3%), 업무용승용차 특례(감가상각 800만/운행기록부) |
| P8 | **성급 단정 + 되묻기 부재** | 정보를 받기 전에 전액 인정/부인 단정, 핵심 질문에 즉답 회피 | 개인/법인, 과세/면세 비중, 금액·시점 등을 먼저 되물어야 |
| P9 | **인정금액·리스크 태그 부적합** | 사안에 성립하지 않는 "인정 0/0원", 산술 오류(예: 150만+150만=225만) | 태그 자체가 무효인 사안 다수 |

## 3. 검증된 법령 레퍼런스 미니 KB (재리서치 절약용)

이 채팅에서 WebSearch로 확인한 사실들. 새 세션에서도 재확인 권장하나 출발점으로 사용 가능.

- **접대비=기업업무추진비**(2024 명칭변경). 중소기업 기본한도 **3,600만원**(일반 1,200만), 수입금액별 100억 이하 **0.3%**(중소 0.36%). **건당 3만원 초과 적격증빙 필수**(경조사비 20만), 미수취 시 손금불산입. 상품권·선물은 신용카드로 구입해야 증빙 확보.
- **직원 명절 선물·떡값·자녀학자금** = 근로소득 과세(원천징수). 소득세법 §12 비과세목록에 없음. 현물도 과세. 자녀학자금은 사내근로복지기금 통하면 비과세.
- **폐업 시 잔존재화** = 부가세법 §10①6호 간주공급. ①과세사업 취득 ②매입세액 공제받은 재화 ③시가 과세. 의원은 면세사업자라 면세진료용(불공제) 재고는 대상 아님.
- **의료보건용역 면세**, 단 **미용목적(성형·피부·비만 지방흡입·필러 등) 과세**. 병의원이 미용시술 하면 과세·면세 **겸영** → 사업자등록 정정·부가세 신고·공통매입세액 안분.
- **면세사업자 매입세액 불공제**(업무용 여부 무관). 임대료 부가세는 필요경비(임차료)로 비용처리 → 소득세 경감. 과세 겸영분만 안분공제.
- **업무용승용차 특례**(법인세법 §27의2/소득세법 §33의2/시행령 §50의2): 일반 승용차가 **대상**(경차·9인승↑·화물·영업용은 제외). 감가상각 연 **800만원 한도**, **운행기록부**로 업무사용비율 산정 → 업무사용분만 손금, 사적사용분 부인(미작성 시 1,500만 기준). 개인 사용차 사업 전입은 시가 계상.
- **재고 폐기/평가손실**: 폐기손실은 폐기물관리법 절차+폐기 객관적 입증(폐기내역서·사진·폐기업체 증명). 파손·부패·유효기간경과 재고 평가손실 = 법인세법 §42③1호·시행령 §78③1호(처분가능 시가 평가).
- **기부금**: 대학 발전기금 = **특례기부금**(법인세법 §24②/소득세법 §34②), 개인사업자 소득 100% 한도 필요경비. 학회 후원 = 공익법인 지정 시 일반기부금(개인 30% 한도) 또는 회비. 특별재난지역 자원봉사 용역 = 특례기부금(봉사일수×8만원+부수 재료비, 기부금확인서).
- **휴업수당**: 사용자 귀책 휴업 = 근로기준법 §46 평균임금 **70% 이상** 지급의무(미지급 3년↓/3천만↓). 무급휴가 일방처리 불가. 4대보험: 국민연금 납부예외(무보수 시)/건강보험 납부유예(1개월↑)/고용·산재 보수총액 기준 — 휴업수당 지급 여부와 연동.
- **육아휴직 대체인력 지원금**(출산육아기 고용안정장려금): 우선지원대상기업 사업주, 육아휴직+대체인력 각 30일↑. 2025년 월 **120만원**. 신청: **고용24 → 기업지원금 → 고용안정장려금 → 대체인력 지원금**. 회계: 수령액 **총수입금액(익금/잡이익) 산입**, 대체인력 인건비는 비용.
- **인플루언서 무료시술 후기**: 세무는 광고효과 입증 시 광고선전비 가능하나 미용시술 무상공급은 시가 부가세·인플루언서 소득 원천징수. **의료법상 대가성 치료경험담=환자 유인·불법 의료광고**(3년↓/3천만↓, 자격정지) — 세무보다 큰 리스크.
- **SaaS/클라우드 구독** = 서비스계약(소유 아님) → 원칙 비용(IFRIC). 구성·맞춤화도 대체로 비용. 소프트웨어 **구매(소유권)**만 자산.
- **자체개발 SW 개발비**: 연구단계=당기비용, 개발단계=자산인식 6요건 충족분만 무형자산(개발비), 상용화 시점부터 상각. 기술적 진보 있으면 R&D 세액공제(조특법 §10, 위탁개발비 포함).
- **리스**: 운용리스=자산 미계상·리스료 전액 손금(단 기간배분), 금융리스=자산+부채 계상·감가상각+이자, 일시불=자산+감가상각. 총 비용은 유사, 차이는 시점·자산계상·부가세·자금부담.
- **진료비 할인**: 실제 받은 금액만 매출(할인분=에누리). 특수관계인 무상·저가는 부당행위계산부인(소득세법 §41) 시가과세. 매출누락(받은 돈 미신고)=탈세, 병의원은 현금영수증 의무발행업종(건당 10만↑ 미발급 20% 가산세).

## 4. 이 채팅에서 리뷰한 세션 (14건, 시간순)

대부분 **법률정확성 1/5, 문장력 1~2/5**. 예외만 표기.

1. 약품 도매상 부도(미지급외상 1,500만·선급금 800만) — 접대비 오적용, 채무↔채권 반대 / 문장력3
2. 추석 직원 떡값 15명×30만 — 근로소득 과세를 "§35 비과세"로 정반대
3. 폐업 시 진료의자·레이저 매각 — 지출 라우팅 실패(P1), 다중턴 맥락붕괴
4. 폐업 재고 5,000만 부가세 — P1 거부(잔존재화 간주공급 사안)
5. 신규채용 청년 고용증대세액공제 — 인건비 30~50% 비율로 산정원리 오류, 2턴 연봉→복리후생비 둔갑
6. 진료비 할인·"빠뜨려도 되나"(매출누락) — 접대비 오적용, 매출누락 경고 누락
7. 직원 자녀 장학금 5명×200만 — "원장 개인 운동" 환각(P2)
8. 산불 의료봉사 비용 — 학회 출장 안분 오적용, 특별재난지역 기부금특례 누락
9. 모교 발전기금 vs 학회 후원 절세비교 — 특례기부금을 접대비로 부인, 비교 미응답
10. 피부과+비만클리닉 업종추가·부가세 — P1 거부

## 6. 새 세션에서 이어가는 법

1. 이 문서 + 메모리 [[feedback_auditor_review_format]] 를 먼저 읽는다.
2. 사용자가 "질문 + AI 답변" 세트를 붙이면 → §1 포맷(3종)으로 반환.
3. 리뷰 시 §2 반복 오류 패턴을 먼저 스캔(P1~P9), §3 미니 KB를 출발점으로 삼되 **WebSearch로 조문·최신 개정 재확인** 후 출처 링크 첨부.
