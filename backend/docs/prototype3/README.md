# Prototype 3 — 감사 워크스페이스 확장 (계정 전환 · 챗 감사 · 지식 베이스)

프로토타입 2의 "헤더 토글" 수준 감사 모드를, **계정 전환 기반의 본격 감사 워크스페이스**로 확장한다.
감사 모드는 더 이상 챗 UI를 빌리지 않고, **데이터 어노테이션 도구**(Outlier 등)에 가까운 3-pane 워크스페이스를 갖는다.

감사 모드 안에는 두 가지 주요 기능이 있다:
1. **챗 로그 감사** — 프로토타입 2에서 만든 라인 피드백 + 세션 평가 (그대로 재사용).
2. **지식 베이스(KB)** — Claude Skill 형식. 마스터 문서 `스킬.md` + 폴더 기반 보조 문서 트리.
   - `해석론/` · `직업군/<업종>/` · `판례노트/` · `용어집/` · `오류패턴/`
   - 파일·폴더명은 모두 한국어. 레포의 기존 docs/data 자산을 시드로 사용.
