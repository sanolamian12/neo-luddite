### 쟁점 택소노미 (세목별 최빈 분쟁)

- **소득세**: 종합소득세 신고·공제(36%), 사업소득 vs 기타소득 구분, 필요경비 인정, 1세대1주택
- **법인세**: 납세의무 성립·신고(35%), 과점주주 2차납세의무(29%), 손금 인정, 부과제척기간
- **부가가치세**: 과세표준(35%), **가공·허위 세금계산서(32%)**, 면세 범위(28%), 매입세액 불공제
- **가산세**: **세금계산서 미발급(80%)**, 정당한 사유, 장기제척 10년 적용
- **부가세 핵심 = 세금계산서 진위**, **소득세 핵심 = 소득구분·필요경비**

## Goal 2 — 결정문 구조

`analysis_structure.py` → `data/analysis/cases_structured.jsonl`, `structure_report.*`

```
조세심판원:  [청구번호][세목][결정유형][결정요지][관련법령]
             → [주문] → [이유] { 처분개요 → 청구인주장 → 처분청의견 → 심리및판단 → 결론 }
유권해석:    질의요지 → 회답 → 이유
대법원:      판시사항 → 판결요지 → 참조조문 → 이유
```

→ 판단 로직 추출의 핵심은 `[이유] > 심리및판단` 섹션 (괄호 라벨 파싱).

### 8대 해석 프레임워크 사용 빈도 (567건 중)

| 프레임워크 | 사용률 | 비고 |
|---|---:|---|
| 실질과세원칙 (§14) | 24% | 최빈 — 거의 모든 사건의 기저 |
| 유추해석 | 19% | 금지원칙이나 역설적 다빈도 |
| 엄격해석 (조세법률주의) | 16% | 감면·비과세 사건의 출발점 |
| 목적론해석 | 13% | 문언 불명확 시 보충 |
| 입증책임 | 11% | 사실관계 분쟁 |
| 문언해석 | 10% | 특정 단어 쟁점 |
| 신의성실원칙 (§15) | 7% | 과세관청 견해표명 사건 |
| 체계적해석 | 1% | 법령 충돌 |

### 프레임워크 선택 결정트리 (핵심 산출물)

```
출발: 엄격해석 선언 (법문대로, 확장·유추 금지)
 1. 문언 명백? → YES: 문언해석 종결
 2. 형식≠실질 (가장행위·조세회피)? → 실질과세원칙(§14)
 3. 법령 충돌? → 체계적해석
 4. 문언이 입법취지 무력화? → 목적론해석(보충)
 5. 과세관청 공적견해+신뢰+불이익? → 신의성실(§15)
```
