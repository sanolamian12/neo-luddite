# 03. 세무 서비스 전략

## 의사결정 2축

1. **세목 선택**: 종합소득세 / 부가가치세 / 법인세 중 무엇?
2. **직업군 타게팅**: 소상공인·프리랜서 → 더 구체적 직업군으로 좁히기

## 결론 1 — 세목: 종소세 + 부가세 (법인세 후순위)

| 세목 | 소상공인·프리랜서 적합도 | 이유 |
|---|:---:|---|
| 종합소득세 | ★★★★★ | 프리랜서 전원 5월 신고, 보편적 페인 |
| 부가가치세 | ★★★★☆ | 사업자등록 소상공인의 반복 페인 |
| 법인세 | ★★☆☆☆ | 타깃 대부분 개인사업자, 법인은 기존 세무사 고착 |

⚠️ **경쟁 현실**: 프리랜서 종소세 *환급*은 삼쩜삼이 장악(레드오션) → 차별화는
**"판단 자동화"**(감면·면세·영세율·업종코드 판별)에 있음.

## 결론 2 — 직업군: 국세통계 기반 정량화

출처: data.go.kr 국세청 100대 생활업종 (2025.8 기준 원본 CSV 집계) + 신종업종 통계

| 직업군 | 모수 | 성장 | 페인 성격 | 침투성 | 전략 위치 |
|---|---:|---:|---|:---:|---|
| **온라인셀러**(통신판매업) | 686,795 | +6.7% | 판단분쟁(영세율·매입세액) | 무주공산 | **1순위 진입** |
| **크리에이터**(1인미디어) | 종소세신고 24,797 (18배↑) | 폭발 | 판단분쟁(감면·면세) | 무주공산 | **1순위 진입** |
| 미용·피부 | 117,935 + 72,311 | +1.8/+5.1% | 정형(현금매출·간이) | 보통 | 2순위 확장(자동화형) |
| 병의원 | 80,861 | +1.3% | 판단(면세안분·비급여) | **요새(레드오션)** | 3순위 프리미엄 |

### 직업군별 핵심 발견

- **온라인셀러**: PG·광고대행 수수료 세금계산서 매입세액 불공제, 매출누락, 구매대행 vs 수출 영세율, 해외직구 관세 화주판단
- **크리에이터**: 창업중소기업 감면 배제(인적용역=사업창출X), 별풍선 면세 여부(직원·설비 보유 시 과세), 업종코드 940306 vs 921505, **2026.4 현금매출명세서 의무화(쐐기)**
- **미용·피부**: 케이스 분쟁 거의 0 → 해석분쟁 아닌 *성실신고(현금매출)* 페인 → "저가 자동화형" 모델에 적합
- **병의원**: 객단가 최고지만 전문 세무사 고착 → 신규 개원의·비급여 위주로 좁게 진입해야

## 전략 순서

```
1순위 진입   : 크리에이터(규제 쐐기) + 온라인셀러(모수)   ← 무주공산, 판단형 프리미엄
2순위 확장   : 미용·피부                                  ← 자동화형 저가 모델
3순위 프리미엄: 병의원(신규개원·비급여로 좁게)            ← 요새, 고가 자문
백엔드 엔진은 공유, 프론트 포지셔닝만 분리
```
